# Research Papers Directory

This directory contains PDF files for research papers. PDF files should be named to match the paper IDs from `lib/content-registry.ts`:

- `cryptocurrency-forecasting-model-paper.pdf`
- `molecule-mutation-prediction-paper.pdf`
- `task-scheduling-paper.pdf`

## Current Files

- `1.pdf` - Please rename to match the appropriate paper ID
- `2.pdf` - Please rename to match the appropriate paper ID

## Adding New Papers

1. Add the PDF file to this directory with the filename matching the paper ID from `content-registry.ts`
2. Update `lib/content-registry.ts` with the new paper entry
3. The PDF will be automatically accessible via the research subdomain
